GEE Zonal Documentation
====================================

**GEE Zonal**: This package contains two classes to ease the process of 
getting statistics from Google Earth Engine datasets.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
